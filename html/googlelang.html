Google

Google is one of the top website out there which is very user interactive. It has lots of features inbuilt and the feature of language change as per the visitor's demand is one of them. There aren't that many people who access google but don't understand English. Althogh, it is a good feature to have in it's arsenal.

This feature increases the engagement of visitors with the website. So in this website, as similiar to the google, the language change feature is added in contact page. If there is any text difficult to understand, visitor can switch the language to Nepali and understand what the text mean. This contact page contains two button specified as English and Nepali where visitor can toggle the text language.

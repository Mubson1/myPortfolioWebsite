Virgin America

Virgin America was a USA airline which operated between 2007 and 2018. The website of Virgin America has beautifully arranged different webpages in a same homepage. A visitor can read a brief description of what the page is about in a box and onclick of the box, it sends the visitor to another webpage that they want to access.

This feature or design is what I would like to say, helps to organize different webpages that are linked or related to the original webpage. In this website, the similiar design is placed where a box is made for the visitor to access different webpage. The box contains a small description and the name of the webpage where it is intended to take.
